(This is also the base URL for all api endpoints.)

#### API Endpoints

**GET `/api/v1/food-trucks`**

- This endpoint will return a list of all food-trucks along with all their attributes.
```json
[
  {
    "data": {
      "attributes": {

      }
    }
  }
]
```
